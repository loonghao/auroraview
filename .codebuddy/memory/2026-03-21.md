# 2026-03-21

## tao/wry WebMCP 调研

- **wry**: 当前 0.54.1，最新 0.54.4（2026-03-17）；**tao**: 当前 0.34.5，最新 0.34.6（2026-03-04）
- WebMCP 是纯 JS 层协议，tao/wry 作为 WebView 容器天然支持，不需要底层改动
- 如果需要 WebMCP 兼容，在 `auroraview-sdk` JS 层实现即可

## 参考项目分析收获（不在代码中引用来源）

- **`pyfuture` crate 模式**: 独立的 Rust Future ↔ Python 异步桥接库，值得参考抽取
- **`pyo3-utils` crate 模式**: 通用 PyO3 工具（py_wrapper, serde, ungil, from_py_dict），值得参考
- **测试策略**: 独立 test crate 模拟完整应用 + Python handler + Rust IPC 闭环测试
- **CI 实践**: `pyright --verifytypes` 验证 Python 类型导出完整性，建议引入
- **commitlint**: commit message 规范化工具，建议引入
