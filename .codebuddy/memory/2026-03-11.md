## CI Pack "bun not found" Fix

- **Root cause**: `gallery/auroraview.pack.toml` has `[build].before = ["bun run build"]` and `[vx].ensure = ["bun"]`. In the pack CLI (`pack.rs`), `before` commands were executed *before* `PackManager.pack()`, but `vx ensure` ran *inside* `pack()` — so bun wasn't installed yet when `bun run build` executed. CI also lacked bun installation.
- **Fix 1 (Rust CLI)**: In `crates/auroraview-cli/src/cli/pack.rs`, added vx ensure execution before `before_build` commands. This ensures tools declared in `[vx].ensure` are available when before commands run (locally with vx).
- **Fix 2 (CI)**: Added `oven-sh/setup-bun@v2` step in `.github/workflows/build-gallery.yml` so bun is available in CI environment.
- **Fix 3 (lib.rs)**: Exported `VxTool` from `auroraview_pack` crate for use by CLI layer.
